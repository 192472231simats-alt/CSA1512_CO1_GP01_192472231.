## Reflection

Zoho Creator is a Software as a Service (SaaS) platform because it allows users to build and use applications through a web browser without installing any software. Zoho hosts the applications on the cloud and manages updates, security, backups, and maintenance. Users can access their applications from anywhere using an internet connection. GitHub provides a cloud-based platform to store project files, track changes, and collaborate efficiently. This guided project improved my understanding of SaaS and the use of cloud platforms for application development and version control.
